# 🤖 Jimmy AI Assistant

> Your personal AI assistant — like FRIDAY from Iron Man, but smarter and more personal!

## ✨ Features

- 🎤 **Voice Control** — "Hey Jimmy" wake word
- 🗣️ **Smart Girl Voice** — Romantic, warm tone
- 🌐 **Hindi + English** — Bilingual support
- 📱 **Phone Controls** — Calls, SMS, Camera, Torch, Alarm
- 🧠 **Claude AI Brain** — Online intelligent responses
- 📶 **Offline Mode** — Works without internet
- 📲 **Installable App** — PWA (installs like real app)
- 🔋 **Battery Check** — Real-time phone info

## 🚀 Setup on GitHub Pages

### Step 1: Create Repository
1. GitHub.com pe jao → New Repository
2. Name rakho: `jimmy-ai`
3. Public raho → Create

### Step 2: Upload Files
Upload these files in order:
- `index.html`
- `style.css`
- `app.js`
- `manifest.json`
- `sw.js`

### Step 3: GitHub Pages Enable Karo
1. Repository → Settings
2. Pages section mein jao
3. Source: `main` branch
4. Save karo

### Step 4: Your App is LIVE! 🎉
```
https://YOUR_USERNAME.github.io/jimmy-ai/
```

### Step 5: Phone pe Install Karo
1. Chrome mein link kholo
2. 3-dot menu → "Add to Home Screen"
3. Install karo — real app ki tarah!
